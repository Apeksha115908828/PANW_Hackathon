# Personalized Goal Forecasting

A minimal full-stack MVP to upload a transactions CSV and a plain-English goal, then check feasibility and offer simple savings suggestions if off-track.

## Tech Stack
- Backend: Java Spring Boot, Maven
- Frontend: React + Vite

## Run Locally

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
